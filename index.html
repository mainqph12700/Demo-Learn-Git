I. repo: dự án (chứa toàn bộ source). 2 loại: local: trên máy-- remote: trên máy khác
II. commit: đc lưu vào repo, mỗi commit là 1 lần sửa, thêm, xóa file (dựa vào commit để truy xuất dev)
III. branch: nhánh
===================
1. git init: create a new repo
2. git clone: clone về máy
3. git pull: đồng bộ từ trên git về máy
4. git add & git add. : thêm file
5. git commit: sửa file
6. git push: đồng bộ từ máy lên repo
7. git log: or git status: check trạng thái file
8. git log --decorate --graph --oneline: check trạng thái file
9. git remote add origin https://github.com/mainqph12700/Demo-Learn-Git.git: gán địa chỉ repo
10.git push -u origin main: push repo local
